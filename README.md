# A collection of bash and powershell scripts

This is a collection of public bash and powershell commands that I have come across or had a need to store that you may also find helpful. Azure Local AZHCI has a number of 'hidden' commands that you can find through trial and error or through Microsoft support cases

In my ventures with Linux, I also come across some commands that may come in helpful in the future and these are stored here too for future reference

# Versioning

Note that there is no versioning on these commands or scripts and they cannot be guaranteed to work on every old and new version of Linux, or Windows

# Security

There are a couple of pipelines configured in this repository that check for typos in the scripts, security recommendations and errors

# Liability

It should come as no surprise, run these scripts at your own risk and only if you know what you are doing
